# Asset licenses — City

This document tracks provenance for 3D/content assets. Godot (MIT) does not
dictate content licenses; each asset must be cleared separately.

## Project code

Unless noted otherwise, project scripts under `scripts/` and tools under
`tools/` (except vendored third-party downloads) are part of this repository.

## Human body bases (MakeHuman / MPFB)

| Asset | Source | License |
|-------|--------|---------|
| `assets/humans/male_base.gltf` (+ `.bin`, `.glb`) | Exported via MPFB (`game_engine` rig) from MakeHuman basemesh | **CC0** (MakeHuman/MPFB core assets) |
| `assets/humans/female_base.gltf` (+ `.bin`, `.glb`) | Same pipeline, female macros | **CC0** |

Generated with:

- MPFB plugin (`tools/vendor/mpfb2-plugin.zip`, nightly 2026-07-21)
- MakeHuman system assets pack (`makehuman_system_assets_cc0.zip`) installed into MPFB user data
- Headless export: `tools/blender_export_mpfb_humans.py` / `tools/export_mpfb_humans.bat`

References:

- https://static.makehumancommunity.org/about/license.html
- https://static.makehumancommunity.org/mpfb/faq/use_in_closed_source.html

## Vendored downloads (not shipped in game)

Under `tools/vendor/` (gitignored): Blender portable, MPFB zip, system asset pack.
These are build-time only.

## Sources we are deliberately not shipping

| Source | Why not |
|--------|---------|
| MB-Lab generated characters | AGPL propagates to outputs |
| SMPL / SMPL-X | Non-commercial free license; commercial needs Meshcapade |
| DAZ Genesis in-game meshes | Interactive License per product |
